Создание розыгрыша:

!gstart %time%amount%title%url%imageUrl

Параметры (* - обязательный):

*time ⏤ время до конца розыгрыша. Измеряется в d(дни), h(часы), m(минуты), s(секунды).
Например, для того, чтобы создать розыгрыш на 2 дня, необходимо написать "2d"
amount ⏤ количество участников
*title ⏤ заголовок розыгрыша
url ⏤ ссылка на предмет розыгрыша
imageUrl ⏤ ссылка на картинку, которая будет вставлена в оповещение о начале и конце

Завершение розыгрыша:

!gend ID

Параметры (* ⏤ обязательный):

*ID ⏤ ID сообщения о начале розыгрыша

Перевыбор участников:

!greroll amount ID

Параметры (* ⏤ обязательный):

*amount ⏤ количество новых победителей
*ID ⏤ ID сообщения о начале розыгрыша

ВНИМАНИЕ! Перевыбрать участников розыгрыша станет невозможным после 4-х суток от конца

Изменение параметров розыгрыша:

!gset %ID%type%data

Параметры (* ⏤ обязательный):

*ID ⏤ ID сообщения о начале розыгрыша
*type ⏤ тип изменяемого параметра
*data ⏤ значение, на которое изменяется параметр

Типы и ожидаемые значение (регистр букв имеет значение):

title ⏤ изменяет название розыгрыша. Строка
time ⏤ изменяет время с начала розыгрыша. Строка
amount ⏤ изменяет количество победителей. Число
url ⏤ изменяет ссылку, которая вставляется в название розыгрыша. Строка
urlImage ⏤ изменяет ссылку на изображение. Строка

Случайный выбор:

!grand %first%second%...

Параметры (* ⏤ обязательный):

first, second и т.д. ⏤ Параметры между которыми будет выбирать бот. Они должны указываться через %
Параметров может быть неограниченное количество
